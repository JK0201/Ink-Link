## Overview
- Ink Link는 캐치마인드와 갈틱폰을 모티브로 개발한 실시간 미니 게임 프로젝트 입니다. 사용자들이 그림을 그리고 맞추는 게임을 통해 소통하고 즐길 수 있는 플랫폼 입니다.

## Outro
  <details>
    <summary><b>느낀점</b></summary>
    <ul>
      <li>로비 페이지와 실시간 기능을 구현하면서 실시간 통신의 복잡성을 경험하고, 이를 해결하는 과정에서 많은 것을 배웠습니다. 실시간으로 데이터를 주고 받는 과정에서 발생하는 다양한 문제들을 해결하기 위해 고민하는 과정은 매우 도전적이였으며, 이를 통해 네트워크와 관련된 기술적 이해도를 높일 수 있었습니다.</li>
      <li>TypeScript를 사용하는 것이 실무나 대규모 프로젝트에서 매우 유익하다는 것을 체감할 수 있었습니다. JavaScript는 동적 타입을 지원해 유연성을 제공하지만, 이로 인해 예상치 못한 타입 관련 오류가 발생할 수 있습니다. TypeScript는 이런 문제를 미리 방지하고 코드의 안정성을 크게 향상시키는 효과를 제공합니다. 이러한 특성 덕분에 프로젝트의 유지보수와 확장성이 대폭 개선되는 것을 실제로 느낄 수 있었습니다.</li>
      <li>개인적인 사정으로 개발 속도가 느려지고 프로젝트가 중단되어 팀원에게 미안하고 아쉬웠지만, 사회생활을 통해 배운 인수인계의 중요성을 느낀 바로서, 중단된 프로젝트의 진행 상황과 각 기능의 구현 방법을 명확하게 기록하는 것이 중요하다고 느꼈습니다. 이에 따라, 언제든지 프로젝트를 효율적으로 재개할 수 있도록 마크다운 문서를 작성했습니다.</li>
    </ul>
  </details>
